## Visão geral

A skill `resume-ats-beater` reformula currículos para maximizar compatibilidade com plataformas ATS e aumentar clareza para recrutadores.

## Quando usar

- Otimizar currículo para ATS
- Adaptar CV para cargo/indústria alvo
- Fortalecer bullets com resultados mensuráveis
- Validar requisitos eliminatórios e aderência semântica
- Preparar currículo para plataformas como Gupy, Vagas.com e Sólides

## Melhorias recentes

**v1.2 (Março 2026)**
- Etapas de diagnóstico sobrepostas (Etapas 2+3) mescladas em uma única etapa unificada
- Templates de saída do diagnóstico movidos para novos arquivos externos
- Adicionado modo padrão explícito (`modo_completo`) quando o modo de execução não é especificado
- Adicionado guardrail: bloquear todo o workflow até que `curriculo_atual` seja fornecido
- Adicionada instrução de skip: `modo_reescrita` ignora a etapa de diagnóstico
- Adicionada nota comportamental para `plataforma_ats_alvo` quando a plataforma é conhecida

## Entradas recomendadas

- histórico profissional
- requisitos da vaga alvo
- links de portfólio e LinkedIn
- restrições de veracidade factual

## Exemplo de uso

```text
Reescreva meu currículo para uma vaga de Product Manager em SaaS B2B mantendo fatos reais e aumentando aderência para ATS.
```

## Visão geral

A skill `ultimate-design-system-master` oferece um workflow com briefing, menu de opções e execução por prompt especializado.

## Quando usar

- Criar design system completo
- Gerar identidade visual e padrões UI/UX
- Produzir assets de marketing
- Traduzir design para código
- Rodar auditoria de acessibilidade

## Como funciona

1. Coleta briefing com questionário estruturado.
2. Apresenta menu com 10 geradores.
3. Executa o prompt de referência selecionado.
4. Retorna ao menu para iteração.

## Exemplo de uso

```text
Quero criar um design system para um produto SaaS B2B com foco em acessibilidade WCAG 2.2 AA e handoff para engenharia.
```

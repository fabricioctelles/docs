## Visão geral

A skill `ultimate-design-system-master` oferece um workflow com briefing, menu de opções e execução por prompt especializado.

## Quando usar

- Criar design system
- Construir identidade de marca
- Gerar padrões UI/UX
- Produzir assets de marketing
- Escrever specs para Figma
- Obter crítica de design
- Analisar tendências de design
- Rodar auditoria de acessibilidade
- Traduzir design para código
- Criar decks de apresentação

## Melhorias recentes

**v2.1 (Março 2026)**
- Descrição reescrita no formato de gatilho em terceira pessoa
- Questionário de Briefing com 18 perguntas movido para arquivo externo
- Adicionado Checklist de Qualidade com condições concretas de verificação
- Removidas introduções redundantes

## Como funciona

1. Coleta briefing com questionário estruturado.
2. Apresenta menu com 10 geradores.
3. Executa o prompt de referência selecionado.
4. Retorna ao menu para iteração.

## Exemplo de uso

```text
Quero criar um design system para um produto SaaS B2B com foco em acessibilidade WCAG 2.2 AA e handoff para engenharia.
```

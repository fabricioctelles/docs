## Instalação rápida

```bash
npx skills add fabricioctelles/skills
```

Para instalar apenas uma skill:

```bash
npx skills add fabricioctelles/skills@coolify-operator
```

<Note>
  Se você ainda não instalou as skills, siga o guia em `/quickstart`.
</Note>

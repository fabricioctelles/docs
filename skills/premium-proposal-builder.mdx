## Visão geral

A skill `premium-proposal-builder` estrutura propostas comerciais com foco em clareza, escopo e conversão.

## Quando usar

- Criar proposta para cliente novo
- Melhorar um pitch já existente
- Montar estrutura de deck ou website de proposta
- Adaptar narrativa para agência, SaaS ou enterprise

## Saída esperada

- estrutura de proposta com seções essenciais
- orientações de formato (website vs deck)
- prompts base para ferramentas de design e apresentação

## Exemplo de uso

```text
Preciso de uma proposta premium para um projeto SaaS B2B. Monte estrutura, escopo, cronograma e pricing.
```

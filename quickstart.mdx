---
title: "Instalação"
description: "Instale as skills do repositório fabricioctelles/skills no seu agente."
---

## Pré-requisitos

- Node.js 18+
- `npx` disponível no ambiente
- Diretório de configuração do seu agente (`.claude/skills`, `.cursor/skills` ou `.kiro/skills`)

## Instalação em três passos

<Steps>
  <Step title="Instale todas as skills">

```bash
npx skills add fabricioctelles/skills
```

  </Step>
  <Step title="Ou instale uma skill específica">

```bash
npx skills add fabricioctelles/skills@premium-proposal-builder
npx skills add fabricioctelles/skills@geo-optimization
npx skills add fabricioctelles/skills@substack-expert
npx skills add fabricioctelles/skills@pier-cloud
npx skills add fabricioctelles/skills@ultimate-design-system-master
npx skills add fabricioctelles/skills@front-end-checklist
npx skills add fabricioctelles/skills@resume-ats-beater
```

  </Step>
  <Step title="Valide o carregamento no agente">

Faça uma pergunta que acione a skill instalada e confirme se o agente aplicou o workflow esperado.

  </Step>
</Steps>

## Alternativa com Agent Skills CLI

```bash
npm install -g agent-skills-cli
skills add fabricioctelles/skills
```

Sem instalação global:

```bash
npx agent-skills-cli install fabricioctelles/skills
```

## Instalação manual

```bash
git clone https://github.com/fabricioctelles/skills.git
```

Depois copie a pasta da skill para o diretório do seu agente.

Exemplo para Claude Code:

```bash
cp -r skills/premium-proposal-builder .claude/skills/
```

<Tip>
  Se você está configurando este projeto pela primeira vez, personalize o arquivo `AGENTS.md` com regras e terminologia do seu time.
</Tip>

<Note>
  Consulte o catálogo em `/skills/index` para escolher a skill mais adequada ao seu fluxo.
</Note>
